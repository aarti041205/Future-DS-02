# 📊 Customer Churn & Retention Analysis

## 🔍 Project Overview

This project focuses on analyzing customer churn behavior using the **Telco Customer Churn Dataset**.
The goal is to identify why customers leave, understand retention patterns, and provide actionable business recommendations to improve customer retention.

---

## 🎯 Objectives

* Analyze customer churn trends and patterns
* Identify high-risk customer segments
* Perform customer segmentation (contract, payment, services)
* Understand customer lifetime behavior
* Provide data-driven recommendations to reduce churn

---

## 🛠️ Tools & Technologies Used

* **Microsoft Excel** – Data cleaning & preprocessing
* **Power BI** – Dashboard creation & visualization
* **Data Analytics Techniques** – Segmentation, trend analysis, KPI analysis

---

## 📁 Dataset

* **Dataset Name:** Telco Customer Churn Dataset
* **Source:** Kaggle
* Contains customer demographics, subscription details, services, and churn status

---

## 📊 Dashboard Features

The Power BI dashboard includes:

### 🔹 Key KPIs

* Total Customers
* Churn Rate (%)
* Retention Rate (%)
* Average Customer Tenure
* Total Churned Customers

### 🔹 Visual Analysis

* Customer Distribution by Churn Status
* Churn Analysis by Contract Type
* Churn by Internet Service Type
* Churn by Payment Method

### 🔹 Interactive Filters

* Gender-based filtering

---

## 🔑 Key Insights

* Customers with **month-to-month contracts** have the highest churn
* **New customers (low tenure)** are more likely to leave
* **Fiber optic users** show higher churn rates
* Customers using **electronic check** are high-risk
* Lack of **tech support & security services** increases churn

---

## 💡 Business Recommendations

* Promote **long-term contracts** to improve retention
* Improve **onboarding experience** for new customers
* Encourage **automatic payment methods**
* Bundle **tech support and security services**
* Optimize pricing for **fiber optic plans**
